Recent Project

Electonic Synthetical Keyboard

  For a recent class, two other friends and I made our own simple synthetic keyboard. We used a microcontroller board called a NUCLEO and other components that were part of our lab kit. Additionally, we bought a pair of small speakers from Amazon. The components for the project included a LCD screen, a digital to analog converter, an amplifier, and push buttons. I mainly worked on the hardware portion of the project with wiring and making an enclosure so our project looked good and did not fall apart. There are 17 keys in total, which equates to around an octave and a half of keys. We used mainly a version of C for the code with an IDE called STM32. There were some improvements we could have made, but ran out of time to complete those extra tasks.
