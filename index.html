Recent Projects

  For a recent class, two other friends and I made our own simple synthetic keyboard. We used a microcontroller board called a NUCLEO and other components that were part of our lab kit. Additionally, we bought a pair of small speakers from Amazon. The components for the project included a LCD screen, a digital to analog converter, an amplifier, and push buttons. I mainly worked on the hardware portion of the project with wiring and making an enclosure so our project looked good and did not fall apart. There are 17 keys in total, which equates to around an octave and a half of keys. We used mainly a version of C for the code with an IDE called STM32. There were some improvements we could have made, but ran out of time to complete those extra tasks.

Hobbies

  I like to build models in my spare time, usually plastic fisgures and robots. I started building around 2016 and kept up with it until now. I like it because at the end, I have something physical that I can either display or mess with. I really like working with physical objects, and building models helps me to keep my hands moving and creating something that I can show off in the end.
  Recently, I started learning how to 3D print using a FDM printer. The printer uses plastic wire and melts the plastic to create layers. A 3D model can be broken up into layers and the printer can take the layers of the model and build it using the melted plastic. I usually just download things off the internet but I began looking into creating some 3D models on my own using CAD, or computer aided design. The program itself allows the user to draw something on a plane and then extrude it to make a 3D model. By using a combination of this and some other tools, I can make models that I designed and then print them to see them in a physical reality.
